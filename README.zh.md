## 🔨 开发者信息

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

### 构建和部署

```bash
# 生产构建
npm run build

# 启动生产服务器
npm start
```

### 项目结构

```
├── app/                 # Next.js App Router 页面
├── components/          # React 组件
├── content/            # Markdown 内容文件
├── lib/                # 工具函数和数据
├── messages/           # 国际化翻译文件
├── public/             # 静态资源
└── styles/             # 全局样式
```
